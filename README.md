# Portfolio 
[일부기능 구현중인 화면][https://amsomad.github.io/portfolio/]
## 뚝딱뚝딱 제작중.

- 현재 적용된 기능
   - geocode 클릭시 해당위치의 주소 알림출력
   - 주소검색 후 해당위치로 이동
   - Vworld 레이어 출력 및 해당레이어 정보 출력
